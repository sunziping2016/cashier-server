    (inner) addDeleted(schema)

    Add a soft-delete hook. It can only be triggered by delete and restore (including both class method and instance method), and is only visible to estimatedDocumentCount, countDocuments, find, findOne, findOneAndUpdate, updateOne, updateMany and their *Deleted and *WithDeleted variants.
